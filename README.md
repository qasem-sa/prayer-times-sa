# 🕌 مواقيت الصلاة - المملكة العربية السعودية

تطبيق مواقيت الصلاة لجميع مناطق ومدن المملكة العربية السعودية

## ✨ المميزات

- 🗺️ 13 منطقة و 100+ مدينة
- 🕌 6 صلوات يومية + الشروق
- 📅 التاريخ الهجري والميلادي
- ⏱️ عداد تنازلي مباشر للصلاة القادمة
- 🕐 نظام 12 ساعة (ص/م)
- 📱 ويدجت للشاشة الرئيسية
- 🌙 تصميم ليلي أنيق
- 📲 PWA قابل للتثبيت بدون متجر
- 💾 حفظ آخر مدينة تم اختيارها

## 🛠️ التقنيات

- HTML / CSS / JavaScript
- PWA (Progressive Web App)
- Aladhan Prayer Times API
- Service Worker للعمل بدون إنترنت

## 📲 التثبيت

### كـ PWA (أسهل)
1. افتح الرابط في Chrome على أندرويد
2. اضغط ⋮ → "إضافة إلى الشاشة الرئيسية"

### كـ APK
```bash
bash setup-android-build.sh
```
